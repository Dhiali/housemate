# Housemate Backend

This is the Node.js + Express backend for the Housemate app.

## Getting Started

1. Install dependencies:
   ```powershell
   cd backend
   npm install
   ```
2. Start the development server:
   ```powershell
   npm run dev
   ```

The server will run on http://localhost:3001 by default.
